## 1.プログラミング言語同士の比較
<br>

**比較対象**
- Python
- C#
- C++
- JavaScript
<br>

**比較内容**
- 動作に最低限必要なコードとコメント
- 標準出力
- 配列
- 変数の定義
- 関数（メゾット）の定義
- クラスの定義
<br>

### 動作に最低限必要なコードとコメント
Python
```Python
#必要なコードはありません。
#ここに書く 
```
C#
```Csharp
using System;
class Program{
    public static void Main(){
        /*
        ここに書く
        */
    }
}
```
C++
```C++
#include <iostream>
using namespace std;
int main(){
    /*
    ここに書く
    */
}
```
JavaScript
```JavaScript
//必要なコードはありません。
```
### 標準出力
Python
``` Python
print("Hello,World!")
```
C#
```Csharp
Console.WriteLine("Hello,World!");
```
C++
```C++
cout << "Hello,World!" << endl;
```
JavaScript
```JavaScript
console.log("Hello,World!");
```
